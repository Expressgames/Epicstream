Playcentrix Code

<script>
const wrap = document.createElement("div");
wrap.style.cssText = "position:fixed;inset:0;z-index:9999";

const ifr = document.createElement("iframe");
ifr.src = "https://epicstreamserver.vercel.app/iframe.html";
ifr.style.cssText = "position:absolute;inset:0;width:100%;height:100%;border:0";

const catcher = document.createElement("div");
catcher.style.cssText = "position:absolute;inset:0;cursor:pointer;background:transparent";

wrap.appendChild(ifr);
wrap.appendChild(catcher);
document.body.appendChild(wrap);

catcher.addEventListener("click", () => wrap.remove(), { once:true });
</script>
Paste this snippet just before the closing </body> tag of your site. It will show the Playcentrix iframe overlay on load and remove it on first click.
